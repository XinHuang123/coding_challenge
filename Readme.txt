Installation:
1. Download and install nodejs at https://nodejs.org/en/
2. Install npm in terminal: 
    npm install
    npm install express
3. Go to the directory and run the terminal: node server_juniper.js
4. Open the page in chrome: http://localhost:3000/public/index.html
